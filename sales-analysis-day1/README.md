# Sales Analysis Using Excel

This project analyzes two-wheeler sales data to understand category-wise sales performance using Excel functions.

## Dataset
The dataset includes:
- Date
- Product
- Category (Bike / Scooter)
- Price
- Quantity
- Total Sales

## Analysis Performed
- Calculated total sales using formulas
- Used SUMIF to find category-wise sales
- Created a column chart to visualize Bike vs Scooter sales

## Tools Used
- Microsoft Excel

## Key Insight
Category-wise analysis helps identify which type of two-wheeler contributes more to overall revenue.
